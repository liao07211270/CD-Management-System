# CD-Management-System
本系統為大專生暑期實習期間所製作，因應檢察署之光碟資料整理需求而開發的光碟內容自動化整合與查詢系統。

<br>

## 開發技術 :
* **開發語言**: Python 3.13.5
* **圖形介面**: Tkinter
* **主要相依套件**: Pillow、pywin32、psutil、reportlab

<br>

## 核心功能 :

系統由兩個可互相切換的子程式組成(需放在同一路徑中)：

### 1. 光碟內容整合工具 (`main.py`)
主要用於標準化光碟備份流程，自動為檔案建立資料夾進行備份。主要功能有以下 :
* 依據年度、分類號、案、卷、目等欄位自動建立規範化的資料夾結構。
* 支援快捷鍵螢幕截圖上傳或從本機匯入圖片。
* 整合完成後，系統會自動在資料夾內生成包含時間、使用者與檔案統計的 `整合報告.txt` 。

### 2. 光碟片搜尋工具 (`cd-searcher.py`)
用於快速查找與管理已備份的光碟檔案。主要功能有以下 :
* 可依據年度、分類號等特定檔號特徵進行精準搜尋。
* 搜尋結果會以顏色標示資料完整度（如檢測結果與光碟內容是否齊全）。
* 支援在系統內直接編輯或刪除資料，並可針對特定年份的資料匯出 PDF 簡易報表]。

<br>

## 如何執行?
本系統預設工作路徑為 `D:\光碟檢測及備份` (若無 D 槽將自動降級至桌面)。
1. 確保已安裝 Python 及 `requirements.txt` 中的相依套件。
2. 執行 `python main.py` 啟動整合工具，或執行 `python cd-searcher.py` 啟動查詢工具。
3. 詳細操作步驟請參閱附帶的 PDF 使用說明書。
