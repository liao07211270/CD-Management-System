# CD-Management-System
本系統是因應檢察署資料整理的需求，因需將舊資料之編號格式更新而開發的光碟內容自動化整合與查詢系統。(使用 Python 與 Tkinter 開發)

## 🛠 開發技術
* [cite_start]**開發語言**: Python 3.13.5 [cite: 863, 1244]
* [cite_start]**圖形介面**: Tkinter [cite: 864, 1245]
* [cite_start]**主要相依套件**: Pillow (圖片處理)、pywin32 (截圖功能)、psutil、reportlab (PDF匯出) [cite: 866, 867, 1247, 1248, 1249]

## ✨ 核心功能

系統由兩個可互相切換的子程式組成：

### 1. 光碟內容整合工具 (`main.py`)
[cite_start]主要用於標準化光碟備份流程，自動為檔案建立資料夾進行備份 [cite: 822]。
* [cite_start]**自動化命名與分類**: 依據年度、分類號、案、卷、目等欄位自動建立規範化的資料夾結構 [cite: 891, 892]。
* [cite_start]**快速截圖與圖片管理**: 支援快捷鍵螢幕截圖上傳或從本機匯入圖片 [cite: 895, 896, 900]。
* [cite_start]**自動產出報告**: 整合完成後，系統會自動在資料夾內生成包含時間、使用者與檔案統計的 `整合報告.txt` [cite: 929, 930]。

### 2. 光碟片搜尋工具 (`cd-searcher.py`)
[cite_start]用於快速查找與管理已備份的光碟檔案 [cite: 1166]。
* [cite_start]**多條件交叉查詢**: 可依據年度、分類號等特定檔號特徵進行精準搜尋 [cite: 1271]。
* [cite_start]**視覺化狀態呈現**: 搜尋結果會以顏色標示資料完整度（如檢測結果與光碟內容是否齊全） [cite: 1277, 1278]。
* [cite_start]**資料夾管理與匯出**: 支援在系統內直接編輯或刪除資料，並可針對特定年份的資料匯出 PDF 簡易報表 [cite: 1280, 1354]。

## 🚀 如何執行
[cite_start]本系統預設工作路徑為 `D:\光碟檢測及備份` (若無 D 槽將自動降級至桌面) [cite: 884, 885, 1265, 1266]。
1. 確保已安裝 Python 及 `requirements.txt` 中的相依套件。
2. 執行 `python main.py` 啟動整合工具，或執行 `python cd-searcher.py` 啟動查詢工具。
3. 詳細操作步驟請參閱附帶的 PDF 使用說明書。
